# djotfmt

[![codecov](https://codecov.io/gh/black-desk/djotfmt/graph/badge.svg?token=GF5SAU9WIC)](https://codecov.io/gh/black-desk/djotfmt)

<div class="warning">

**This project is under very early stage and not ready for use.**

</div>

This project is a [`djot`](https://djot.net) formatter written in Rust.

<div id="Build" class="section">

## Build

``` bash
cargo build --release
```

</div>

<div id="Test" class="section">

## Test

``` bash
cargo test --release
```

</div>

<div id="Install" class="section">

## Install

``` bash
cargo install --path .
```

</div>

<div id="Usage" class="section">

## Usage

You can check the usage by running:

``` bash
cargo run -r -- --help
```

Or after installing:

``` bash
djotfmt --help
```

</div>
